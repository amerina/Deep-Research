# A 股研究报告结构模板

根据任务模式选择对应结构。默认优先使用**单家公司研究报告**，对比报告仅在明确比较时使用。

---

## 一、单家公司研究报告

```markdown
# [公司名]（[股票代码]）研究结论

## 一句话结论

[一句话说清：公司质地、当前估值、最重要的看点与风险]

## 1. 核心看点

1. **[看点1]**：[对应的已披露事实或研究推导]
2. **[看点2]**：[说明]
3. **[看点3]**：[说明]

## 2. 公司是怎么赚钱的

### 2.1 商业模式
[卖什么、客户是谁、利润来自哪里]

### 2.2 行业位置
[行业空间、竞争格局、公司定位]

## 3. 财务与经营质量

### 3.1 成长性
[收入、归母、扣非、ROE/ROIC 趋势]

### 3.2 财务质量
[毛利率、净利率、经营现金流、资产负债表、应收/存货等]

### 3.3 股东回报
[分红、回购、资本开支、融资与减持情况]

## 4. 估值判断

- **估值口径**：[PE / PB / 股息率 / EV/EBITDA]
- **估值日期**：[YYYY-MM-DD]
- **当前判断**：[便宜 / 合理 / 偏贵]
- **市场在交易什么**：[高分红 / 产能释放 / 周期反转 / 事件催化]

## 5. 核心风险

1. **[风险1]**：[说明]
2. **[风险2]**：[说明]
3. **[风险3]**：[说明]

## 6. 催化剂与跟踪指标

- [指标1]
- [指标2]
- [指标3]

## 7. 最终判断

### 7.1 适合什么逻辑
[红利 / 价值 / 成长 / 周期 / 事件]

### 7.2 不适合什么前提
[在哪些假设下结论会失效]

## 参考资料

### 一手资料（L1/L2）
1. [标题](链接) - [说明]
2. ...

### 辅助资料（L3/L4）
1. [标题](链接) - [说明]
2. ...
```

## 四、交付检查清单

- [ ] 一句话结论可直接复述
- [ ] 核心结论有事实支撑
- [ ] 区分了已披露事实、管理层表述、研究推导
- [ ] 估值口径与日期已明确
- [ ] 风险没有被弱化或遗漏
- [ ] 若涉及分红，已区分拟分红与已实施分红
- [ ] 若是对比，已说明可比性边界
