# A 股研究中间产物模板

这些模板仅在**留痕模式**下使用。默认轻量模式下，可以不保存中间文件，直接在对话中完成研究与输出。

## 00_问题拆解.md

```markdown
# 问题拆解

## 原始问题
[用户提出的问题]

## 研究模式
[单家公司深度研究 / 公司对比研究]

## 研究对象

- **公司**：[公司名 / 股票代码]
- **对比对象**：[如无则留空]
- **时间点**：[例如：截至 YYYY-MM-DD]
- **研究目标**：[价值 / 成长 / 周期 / 事件 / 综合判断]

## 研究边界

| 维度 | 边界定义 | 说明 |
|------|----------|------|
| **市场** | [A股主板 / 创业板 / 科创板 / 北交所] | 明确上市地与监管口径 |
| **时间** | [近 3 年 / 近 5 年 / 最新财报口径] | 财务与估值观察窗口 |
| **口径** | [归母 / 扣非 / TTM / 静态 / 已实施 / 拟分红] | 避免混用 |
| **研究层级** | [公司层面 / 业务分部 / 股东回报 / 估值] | 限定范围 |

## 子问题列表

1. 公司靠什么赚钱，业务结构是否清晰？
2. 行业位置、竞争优势、核心变量是什么？
3. 财务质量、成长性、现金流和股东回报如何？
4. 当前估值反映了什么预期？
5. 关键风险、催化剂、后续跟踪指标是什么？

## 是否需要留痕文件

- [ ] 不需要，直接输出最终结论
- [ ] 需要，保存事实卡片与推导过程
- [ ] 需要完整底稿并可打包
```

## 01_资料来源.md

```markdown
## 资料 #[序号]

- **标题**：[资料标题]
- **链接**：[URL]
- **层级**：L1 / L2 / L3 / L4
- **发布日期**：[YYYY-MM-DD]
- **时效状态**：✅当前有效 / ⚠️需验证 / ❌已过时
- **适用对象**：[公司 / 业务 / 板块 / 同行]
- **研究用途**：[商业模式 / 财务 / 分红 / 估值 / 风险 / 公司对比]
- **与研究边界匹配**：✅完全匹配 / ⚠️部分重叠 / 📎仅作补充
- **摘要**：[1-2 句关键内容]
- **与子问题关联**：[对应哪个子问题]

## A 股资料优先级提醒

- L1：年报、半年报、季报、招股书、临时公告、问询函回复、权益分派公告
- L2：投资者关系活动记录、业绩说明会、公司官网
- L3：主流财经媒体、行情聚合页、研报摘要
- L4：股吧、论坛、自媒体
```

## 02_事实卡片.md

```markdown
## 事实 #[序号]

- **主题**：[商业模式 / 成长性 / 财务质量 / 分红 / 估值 / 风险 / 催化剂]
- **类型**：[已披露事实 / 管理层表述 / 研究推导]
- **陈述**：[具体事实或判断]
- **出处**：[资料#序号] [链接]
- **日期口径**：[YYYY-MM-DD 或 2024年报 / 2025Q1 等]
- **适用对象**：[公司整体 / 某业务 / 对比公司]
- **置信度**：✅高 / ⚠️中 / ❓低
- **备注**：[是否涉及拟分红、静态PE、TTM 等口径提醒]
```
