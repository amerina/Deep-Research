# A 股事实卡片模板

用于记录 A 股公司研究中的关键事实。优先服务单家公司研究，也可用于公司对比。

## 模板格式

```markdown
## 事实卡片 #[编号]

**主题**：[商业模式 / 行业位置 / 成长性 / 财务质量 / 股东回报 / 估值 / 风险 / 催化剂]

**类型**：[已披露事实 / 管理层表述 / 研究推导]

**事实陈述**：
[使用陈述句写清楚事实，不带模糊评价]

**出处**：
- 来源：[年报 / 季报 / 招股书 / 公告 / IR 记录 / 行情页 / 媒体]
- 链接：[URL]
- 章节：[页码 / 章节 / 公告标题]
- 获取日期：[YYYY-MM-DD]

**日期/口径**：
- 日期：[YYYY-MM-DD / 2024 年报 / 2025Q1]
- 口径：[归母 / 扣非 / TTM / 静态 / 已实施 / 拟分红 / PB / PE]

**置信度**：[✅ 高 / ⚠️ 中 / ❓ 低]

**置信度理由**：
- ✅ 高：年报、公告、监管披露、公司明确披露
- ⚠️ 中：管理层表述、IR 记录、L3 资料交叉验证
- ❓ 低：媒体转述、社区讨论、无法闭环验证

**适用对象**：
[公司整体 / 某业务板块 / 同行业对比对象]

**相关事实**：
[关联的其他事实卡片编号]

**备注**：
[是否涉及异常项、一次性损益、并表变化、拟分红、估值日期等]
```

## 推荐优先记录的事实

1. 近 3-5 年收入、归母、扣非、经营现金流
2. 主要业务与毛利结构
3. 产能、销量、价格、订单、客户结构
4. 分红方案、回购、再融资、减持、股权激励
5. PE / PB / 股息率及日期口径
6. 风险事件：问询函、处罚、诉讼、减值、关联交易

## 使用示例

```markdown
## 事实卡片 #008

**主题**：股东回报

**类型**：[已披露事实]

**事实陈述**：
公司 2024 年度利润分配预案为每 10 股派发现金红利 20 元（含税）。

**出处**：
- 来源：公司年度报告 / 利润分配预案公告
- 链接：https://example.com
- 章节：利润分配预案
- 获取日期：2026-04-15

**日期/口径**：
- 日期：2024 年报
- 口径：拟分红

**置信度**：✅ 高

**置信度理由**：
来自公司正式披露文件。

**适用对象**：
公司整体

**相关事实**：
#003, #012

**备注**：
拟分红不等于已实施分红，计算股息率时需标注为预案口径。
```

## 矛盾信息记录

```markdown
## 矛盾 #M001

**事实 A**：[来源 A 的说法]
**事实 B**：[来源 B 的说法]

**来源对比**：
- A 来自：[L1 / L2 / L3 / L4]
- B 来自：[L1 / L2 / L3 / L4]

**问题类型**：
[数字不一致 / 日期不一致 / 口径不一致 / 已实施 vs 拟披露]

**处理方式**：
- [ ] 优先采信更高层级来源
- [ ] 保留分歧并在报告中说明
- [ ] 继续补充验证

**验证方案**：
[下一步如何确认]
```
